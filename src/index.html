Node.js и Npm Обзор Node.js - это интерпретатор языка JavaScript. Сам по себе Node.js является C++ приложением, которое получает на входе JavaScript-код и выполняет его.
NPM (Node.js Package Manager) — менеджер пакетов, входящий в состав Node.js, который используется для скачивания пакетов из облачного сервера npm, либо для загрузки пакетов на эти сервера.
Пакетом в Node.js называется один или несколько JavaScript-файлов, представляющих собой какую-то библиотеку или инструмент.
Команды
node - запуск скрипта
npm init – создает package.json в текущей директории
npm install – установка пакетов из package.json
npm install - установка пакета
npm run - запуск команды из package.json
npm search – все доступные пакеты для установки и их краткое описание (можно выполнять выборочный поиск)
Package.json и скачивание пакетов Файл package.json содержит в себе информацию о вашем приложении: название, версия, автора, зависимости и так далее. Любая директория, в которой есть этот файл, интерпретируется как Node.js-пакет, даже если вы не собираетесь публиковать его. Для скачивания пакета файл package.json не обязателен, обычная команда скачает библиотеку в текущую директорию.
